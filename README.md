# ImageTransitionAnimationsUWP

Image control with transition animations. Animation starts when ImageUri property is changed.

## Animation types
* Opacity,
* OpacitySpring,
* ScaleAndOpacity,
* SlideHorizontally,
* SlideVertically,
* StackFromLeft,
* StackFromRight,
* StackFromTop,
* StackFromBottom,
* StackAndScaleFromLeft,
* StackAndScaleFromRight,
* StackAndScaleFromTop,
* StackAndScaleFromBottom

## Properties
* AnimationType
* Direction
* Duration
* DecodePixelHeight
* DecodePixelWidth
* ImageUri
* Stretch
